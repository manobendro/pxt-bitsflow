# MPU6050

Read **acceleration**, **rotation** and **temperature** from an InvenSense **MPU6050**
(accel + gyro) over I2C. Built on the [I2C](/reference/i2c) bus — pure software, so it
works with the existing firmware.

Wire `SDA`/`SCL` to the I2C pins (default `SDA=P4`, `SCL=P5`); call `i2c.init(sda, scl)`
first if you use other pins. The address is **104** (`0x68`), or **105** (`0x69`) if
the `AD0` pin is high.

## init

Wake the sensor from sleep. Call once before reading.

```sig
mpu6050.init(104)
```

### Parameters

* **address**: 7-bit I2C address, `104` (`0x68`) or `105` (`0x69`).

## accelX / accelY / accelZ

Acceleration on each axis, in g (1 g ≈ 9.81 m/s²), at the default ±2 g range.

```sig
mpu6050.accelX()
```

## gyroX / gyroY / gyroZ

Rotation rate on each axis, in degrees/second, at the default ±250 °/s range.

```sig
mpu6050.gyroZ()
```

## temperature

Chip temperature in degrees Celsius.

```sig
mpu6050.temperature()
```

## Example

Show the X acceleration on an OLED, updating ten times a second.

```blocks
oled.init(128, 64, 60)
mpu6050.init(104)
basic.forever(function () {
    oled.showString("aX: " + mpu6050.accelX(), 0)
    basic.pause(100)
})
```
