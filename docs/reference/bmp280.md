# BMP280

Read **temperature** and **pressure** from a Bosch **BMP280** sensor over I2C. Built on
the [I2C](/reference/i2c) bus — pure software, so it works with the existing firmware.

Wire `SDA`/`SCL` to the I2C pins (default `SDA=P4`, `SCL=P5`); call `i2c.init(sda, scl)`
first if you use other pins. The address is **118** (`0x76`) on most modules, **119**
(`0x77`) on some.

## init

Initialize the sensor (reads its factory calibration). Call once before reading.

```sig
bmp280.init(118)
```

### Parameters

* **address**: 7-bit I2C address, `118` (`0x76`) or `119` (`0x77`).

## temperature

Read the temperature in degrees Celsius.

```sig
bmp280.temperature()
```

## pressure

Read the pressure in pascals (Pa). Divide by 100 for hPa / millibar.

```sig
bmp280.pressure()
```

## altitude

Estimate altitude in metres from the pressure and a sea-level reference.

```sig
bmp280.altitude(1013)
```

### Parameters

* **seaLevelhPa**: sea-level pressure in hPa (default `1013.25`).

## Example

Show temperature and pressure on an OLED once a second.

```blocks
oled.init(128, 64, 60)
bmp280.init(118)
basic.forever(function () {
    oled.showString("T: " + bmp280.temperature() + " C", 0)
    oled.showString("P: " + (bmp280.pressure() / 100) + " hPa", 1)
    basic.pause(1000)
})
```
